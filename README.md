# kubectl

[![Build](https://github.com/gsmlg-ci/kubectl/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/kubectl/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/kubectl`
- `ghcr.io/gsmlg-dev/kubectl`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/kubectl:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/kubectl:latest
```

## Build

```bash
docker build -t gsmlg/kubectl .
```
